# CISK Spawn Mod 1.0.19 Notes

Based on 1.0.18, continuing the lightweight RPG layer while preserving the 1.0.12 Sable/Create seat branch.

## Added / Expanded

- Expanded Geera's fishing tips with Starcatcher-style hints.
- Added more Scoria foreshadowing in Geera dialogue.
- Added Geera shop-day workstation behavior prototype:
  - on shop days she can look for nearby barrels/chests/crates/tables/water and walk toward them as a lightweight routine.
  - this is intentionally simple and safe, not a full villager schedule.
- Added more Geera + Mortimer couple banter.
- Added more Mortimer crew log information about optional/lightweight RPG design.
- Added more Scoria advanced-engineering foreshadowing in Mortimer's late quest dialogue.
- Added more travel lines referencing navigation drift, Scoria's engineering knowledge, and Mortimer's fear of repeat mistakes.
- Added stronger logbook feedback after the first Mortimer quests.

## Still Experimental

- Physical schedules are still simple anchor behaviors, not full daily pathing.
- Scoria is still dialogue/logbook groundwork only; no entity yet.
- Geera's shop is still text/status only; no trade GUI yet.
- Create/Sable seating remains based on the 1.0.12 branch, which was the best integration point so far.
