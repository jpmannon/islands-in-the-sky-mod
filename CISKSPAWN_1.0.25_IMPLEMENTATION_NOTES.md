# 1.0.25 Implementation Notes

This build continues the lightweight RPG systems from 1.0.24.

## Added
- Scoria placeholder entity: `ciskspawn:scoria`.
- Scoria renderer/model/assets using temporary Geera-derived placeholder files.
- Scoria UI actions: engineering lesson, Mortimer family clue, reveal clue, crew log, mood bubble.
- Scoria persistent reveal/clue/lesson counters.
- Geera manual bait-and-tackle station anchor.
- Geera station page actions: set station here, call to station.
- Geera shop payment prototype: bait now costs one copper ingot or one emerald.
- Mortimer manual Guild anchor.
- Mortimer Guild page actions: set guild point, call to guild point.
- Expanded UI routing for Scoria, Geera workstation, and Mortimer Guild status pages.

## Still placeholder / future polish
- Scoria needs his real model, texture, and animations.
- Geera shop is still not a true Minecraft container menu; it is a styled dialogue/shop prototype.
- Mortimer Guild visits now have a manual physical anchor, but not a full Guild hall or crew NPCs yet.
- Seating/mounting polish remains separate and should be handled after the RPG systems stabilize.
